# Safety and Compliance

## Safety Rules

- No diagnosis statements
- No treatment recommendations
- Uncertainty must be explicit
- Refer users to clinicians for decisions

## Security Baseline

- Encryption at rest and in transit
- Signed upload URLs
- Role-based access control
- Minimal PII retention

## Auditability

- Store prompt/version hashes
- Store source references per generated statement
- Maintain event logs for report processing
