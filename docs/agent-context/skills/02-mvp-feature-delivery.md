# Skill 02: MVP Feature Delivery

## Goal
Ship MVP features with production-grade structure and safety constraints.

## Steps
1. Read relevant sprint + ticket acceptance criteria
2. Confirm architecture path (interfaces, factories, settings, workflows)
3. Implement feature behind typed contracts
4. Add unit/integration tests as needed
5. Update product and architecture docs

## Guardrails
- No hardcoded secrets/endpoints in code
- Respect feature flags for staged rollout
- Keep commits cherry-pick safe
