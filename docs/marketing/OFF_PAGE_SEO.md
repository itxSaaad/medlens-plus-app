# Off-page SEO checklist

Operational launch gate for MedLens+ marketing surfaces. Not automated in CI.

## Search console

- [ ] Verify domain in Google Search Console (DNS or meta tag)
- [ ] Verify domain in Bing Webmaster Tools
- [ ] Submit `https://medlens.plus/sitemap.xml` after deploy
- [ ] Monitor index coverage weekly during public beta

## Launch channels

- [ ] Product Hunt listing prepared (screenshots, tagline, maker comment)
- [ ] BetaList / alternative beta directories (factual copy only)
- [ ] Health-tech directories where appropriate (no diagnostic claims)

## Backlinks and authority

- [ ] Link from GitHub repository README to production site
- [ ] Guest posts on lab literacy topics (link to glossary hub)
- [ ] Open-source contributors page links to `/about` and `/security`
- [ ] Consistent `Organization.sameAs` social profiles when live

## Brand consistency

- [ ] Same product name "MedLens+" across listings
- [ ] Canonical URL `https://medlens.plus` in all external profiles
- [ ] Privacy and security pages linked from app store / directory listings

## Post-launch monitoring

- [ ] Track branded search "MedLens+" in Search Console
- [ ] Spot-check AI answers (Perplexity, ChatGPT) for accurate product description
- [ ] Review referral traffic in GA4 after GTM consent rollout
