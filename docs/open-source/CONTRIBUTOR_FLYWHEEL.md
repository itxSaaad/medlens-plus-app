# Contributor Flywheel

## Why People Will Contribute

- Real-world healthcare AI problem
- Clear ticketing and architecture
- Strong portfolio value
- Public learning artifacts

## How To Attract Contributors

- Publish roadmap and "good first issue"
- Share weekly build logs on LinkedIn/X/Reddit
- Post contribution calls in OSS communities
- Review PRs quickly and respectfully

## Where To Find Contributors

- GitHub topics and issue boards
- Reddit: r/opensource, r/webdev, r/learnprogramming
- Discord communities (Next.js, Python, OSS)
- Dev.to and Hashnode posts
- LinkedIn engineering groups
