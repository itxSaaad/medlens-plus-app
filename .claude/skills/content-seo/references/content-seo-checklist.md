# Content SEO checklist

Use with `content-seo` when creating or revising indexable pages.

## Intent & keywords
- [ ] One primary intent per page documented
- [ ] Primary keyword in title, H1, URL slug, opening paragraph
- [ ] No keyword cannibalization with existing pages
- [ ] Meta description unique and click-worthy

## Structure
- [ ] Logical H2/H3 hierarchy
- [ ] Hub page links to spokes; spokes link back to hub
- [ ] 3+ contextual internal links with descriptive anchors
- [ ] CTA matches intent (learn vs sign up)

## E-E-A-T
- [ ] About / privacy / security linked where relevant
- [ ] Author or team attribution on long-form content
- [ ] Facts verifiable; sources linked for medical-adjacent claims
- [ ] “Not medical advice” where appropriate

## YMYL / MedLens
- [ ] No diagnostic or prescription language
- [ ] No fear-mongering headlines
- [ ] Cross-lab caution mentioned when discussing comparison
- [ ] `safety-privacy` pass on new copy

## Handoff
- [ ] `technical-seo` — metadata, canonical, sitemap, schema
- [ ] `answer-engine-optimization` — FAQ blocks, definitional lead, llms.txt
- [ ] `ui-accessibility` — readable, semantic markup
